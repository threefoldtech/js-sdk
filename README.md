Threefold Grid extension based on [js-ng framework](https://github.com/threefoldtech/js-ng)

## Contribution

- Clean code (pep-8)
- Documentation
- Tests

## Development environment

- clone repo
- `poetry update` if you already had an install before
- `poetry install`
- `poetry shell`

## Running tests

- `make tests`

## Generating docs

- `make docs`


## building dists

- `poetry build`

## publishing

- `poetry publish`
